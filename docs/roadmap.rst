Roadmap
=======

See the `ROADMAP.md <https://github.com/lucaswiman/revex/blob/master/ROADMAP.md>`_
file in the repository for the full development roadmap, including:

- Near-term improvements (Unicode, flags, API)
- Integration opportunities with Hypothesis and static analysis tools
- Security applications
- Longer-term research directions (symbolic DFAs, regex simplification,
  context-free grammar extensions)
